# بنيان (Bunean) - تطبيق البناء والمقاولات

> Progressive Web App شاملة لقطاع البناء والمقاولات في العراق

---

## ملخص المشروع

بنيان هو تطبيق ويب تقدمي (PWA) متكامل لقطاع البناء والمقاولات يوفر:
- متجر مواد بناء
- صفحة أفكار وتصاميم
- طلبات استشارات وعروض أسعار
- حسابات شركات ومستخدمين
- نظام إشعارات
- تتبع الطلبات
- دعم الوضع الليلي والنهاري
- يعمل بدون إنترنت (offline)

---

## تفاصيل الملفات

### الصفحة الرئيسية (index.html)
- **الغرض**: صفحة تسجيل الدخول / التسجيل
- **المميزات**:
  - تسجيل الدخول برقم الهاتف مع OTP
  - كود OTP تجريبي: أي 4 أرقام
  - اختيار نوع الحساب: مستخدم أو شركة
  - حفظ بيانات الحساب في localStorage
  - تسجيل الخروج

### الصفحة الرئيسية (home.html)
- **الغرض**: الصفحة الرئيسية للتطبيق
- **المميزات**:
  - سلايدر صور رئيسي مع تمرير تلقائي
  - شريط البحث للبحث عن مشاريع ومقاولين ومنتجات
  - 11 قسم للتصفح (مشاريع سكنية، تجارية، مكاتب، مستودعات، حدائق، دور عبادة، مدارس، طرق، جسور، مشاريع حكومية)
  - عرض الشركات المميزة
  - معرض المشاريع المنجزة
  - زر إضافة مشروع جديد

### صفحة الأفكار (ideas.html)
- **الغرض**: استكشاف الأفكار والتصاميم
- **المميزات**:
  - 9 فلاتر أقسام (تصميم داخلي، خارجي، معماري، مناظر طبيعية، تنسيق حدائق، ديكور، تجاري، صناعي، زراعي)
  - عرض المشاريع بالشبكة
  - تفاصيل المشروع مع معرض صور
  - عرض المواد المستخدمة
  - إمكانية الحفظ والمشاركة

### صفحة الاستشارات (quotes.html)
- **الغرض**: إدارة الاستشارات وطلبات عروض الأسعار
- **المميزات**:
  - نموذج طلب استشارة
  - نموذج طلب عرض سعر
  - إرسال النماذج وحفظها في localStorage
  - عرض رسالة نجاح بعد الإرسال

### صفحة طلب الخدمة (service-request.html)
- **الغرض**: نموذج طلب خدمة مفصل
- **المميزات**:
  - اختيار نوع الخدمة
  - إدخال تفاصيل المشروع
  - رفع ملفات
  - إرسال الطلب

### صفحة ملف الشركة (company.html)
- **الغرض**: ملف الشركة (عرض المشرف/التعديل)
- **المميزات**:
  - معلومات الشركة (اسم، وصف، تقييم، مشاريع، موقع)
  - معرض المشاريع المنجزة
  - إحصائيات الشركة
  - أزرار التواصل (رسالة، مكالمة)
  - عرض معرض الصور بالشاشة الكاملة
  - تفاصيل المشروع مع المواد المستخدمة
  - تعديل معلومات الشركة

### صفحة التنفيذ (execute.html)
- **الغرض**: صفحة خدمات التنفيذ
- **المميزات**:
  - عرض خدمات التنفيذ المتاحة
  - طلبات عروض الأسعار
  - إدارة الطلبات

### صفحة السوق (market.html)
- **الغرض**: متجر مواد البناء
- **المميزات**:
  - **أقسام المنتجات**: 8 أقسام (مواد انشائية، اصباغ وديكور، صحيات، كهرباء، أبواب ونوافذ، تكييفات، حدائق، أدوات ومعدات)
  - **عرض المنتجات**: عرض أفقي وشبكي
  - **المنتجات الأكثر طلباً**: تصفية حسب التقييم 5 نجوم
  - **عروض خاصة**: منتجات بخصومات
  - **الموردون المعتمدون**: عرض الموردين
  - **سلة المشتريات**: إضافة/حذف/تعديل الكمية
  - **إتمام الشراء**: عنوان → طريقة الدفع → طريقة التوصيل → تأكيد
  - **الحاسبة**: حساب المساحات والأحجام
  - **تفاصيل المنتج**: ألوان، تشطيب، حجم، كمية
  - **عرض التفاصيل**: زر في كل بطاقة منتج
  - **إخفاء القوائم**: عند فتح تفاصيل المنتج تختفي القوائم
  - **تصفح الصور**: سحب للتنقل، تقريب باللمس

### صفحة المنتج (product.html)
- **الغرض**: صفحة منتج مستقلة
- **المميزات**:
  - عرض صورة المنتج
  - اختيار اللون (5 ألوان)
  - اختيار الحجم (3 أحجام)
  - تحديد الكمية
  - ملخص الطلب
  - زر أضف للسلة
  - زر اطلب الآن

### صفحة الحساب (account.html)
- **الغرض**: إدارة حساب المستخدم
- **المميزات**:
  - **المستخدم**: الملف الشخصي، إعدادات الحساب، الإشعارات، طلباتي، عروضي، المفضلة
  - **الشركة**: إدارة المشاريع، الإعلانات، عروض الأسعار، طلبات الخدمة
  - تعديل معلومات الحساب
  - تبديل الوضع الليلي/النهاري
  - تسجيل الخروج
  - رفع صور الأعمال (حتى 5 صور)
  - إدارة المشاريع المنجزة

### صفحة البروفايل (profile.html)
- **الغرض**: عرض ملف الشركة العام
- **المميزات**:
  - معلومات الشركة
  - معرض المشاريع
  - التقييمات
  - أزرار التواصل

### صفحة الإشعارات (notifications.html)
- **الغرض**: إدارة الإشعارات
- **المميزات**:
  - **5 تبويبات**: الدعم الفني، العروض، طلب الخدمة، طلبات العروض، طلب الخدمة (للشركات)
  - عرض الإشعارات مع تفاصيل كل إشعار
  - تحديث حالة القراءة
  - الرد على الإشعارات

### صفحة تتبع الطلب (order-tracking.html)
- **الغرض**: تتبع حالة الطلب
- **الممونيات**:
  - **خطوات التتبع**: 5 مراحل (تم استلام الطلب، قيد المراجعة، تم التأكيد، قيد التنفيذ، تم التسليم)
  - تفاصيل المنتج
  - كشف الحساب
  - فاتورة PDF

### صفحة تفاصيل الطلب (request-detail.html)
- **الغرض**: عرض تفاصيل طلب الخدمة
- **المميزات**:
  - حالة الطلب
  - تفاصيل الخدمة
  - الملفات المرفقة

### ملفات CSS/SCSS

#### المتغيرات (_variables.scss)
- **الألوان الأساسية**:
  - `$primary-dark`: #1a3a5c (الأزرق الداكن - الخلفية الرئيسية)
  - `$primary-blue`: #2563eb (الأزرق - الأزرار)
  - `$accent-gold`: #f59e0b (الذهبي - التمييز)
  - `$accent-orange`: #f97316 (البرتقالي - التحذيرات)
- **النصوص**:
  - `$text-white`: #ffffff
  - `$text-light`: #8a9bb0
  - `$text-muted`: #5a7088
- **الخلفيات**: إصدارات داكنة وفاتحة لكل لون
- **الحدود**: شفافة وخفيفة
- **الاستدارة**: sm (8px)، md (12px)، lg (16px)، xl (20px)، full (9999px)
- **أحجام الخط**: xs (11px) إلى 4xl (24px)
- **الظلال**: sm، md، lg
- **أحجام الأيقونات**: sm (16px)، md (20px)، lg (24px)

#### المزججات (_mixins.scss)
- `flex-center`: فلكس في المركز
- `flex-between`: فلكس بين
- `flex-column`: فلكس عمودي
- `card`: تنسيق البطاقات
- `input`: تنسيق حقول الإدخال
- `btn-primary`: زر رئيسي
- `btn-outline`: زر هامشي
- `badge`: شارة
- `scrollbar-hide`: إخفاء شريط التمرير
- `glass-effect`: تأثير الزجاج
- `responsive`: استجابة

### ملفات JavaScript

#### header.js
- `renderHeader(options)`: إنشاء الهيدر
  - `searchVModel`: نموذج البحث
  - `searchPlaceholder`: نص البحث
  - `showBack`: إظهار زر الرجوع
  - `backLink`: رابط الرجوع
  - `cartBadge`: شارة السلة
  - `title`: العنوان
  - `notifs`: الإشعارات

#### bottom-nav.js
- `renderNav(activePage)`: إنشاء شريط التنقل السفلي
  - 5 عناصر: الرئيسية، تنفيذ، سوق البناء، أفكار، حسابي
- `requireAccount(callback)`: التحقق من تسجيل الدخول

#### nav.js
- `loadFromStorage(key, defaultVal)`: قراءة آمنة من localStorage
- `initTheme()`: تحميل وتطبيق الثيم
- `toggleTheme()`: تبديل الوضع الليلي/النهاري
- `showBrandAlert()` / `closeBrandAlert()`: نظام التنبيهات
- `goHome()`: الذهاب للرئيسية
- `showTab(tabId, el)`: تبديل التبويبات
- `openMenu()` / `closeMenu()`: فتح/إغلاق القائمة الجانبية
- `initFloatingCart()`: السلة العائمة عبر الصفحات

#### companies.js
- `img()`: إرجاع صورة عشوائية من المجموعة (img1-img31)
- `makeProjects()`: إنشاء بيانات المشاريع
- `makeWorkImages()`: إنشاء بيانات صور الأعمال
- `getCompanyById(id)`: البحث عن شركة بالرقم
- 10 شركات جاهزة (5 مقاولات + 5 ديكور)

#### project-modal.js
- `createProjectModal(opts)`: نظام المودال القابل لإعادة الاستخدام
  - لمس للتنقل بين الصور
  - سحب للإغلاق
  - نقر مزدوج للتقريب
  - تقريب بالضغط
  - سحب عند التقريب

## التغييرات والتعديلات

### التعديلات الأولى
1. **استبدال أزرار السلة بعرض التفاصيل**:
   - تم استبدال زر "أضف للسلة" بزر "عرض التفاصيل" في:
     - المنتجات الأكثر طلباً
     - صفحة عرض كل المنتجات
     - عروض خاصة (أفقي + شبكي)
     - عرض منتجات القسم (أفقي + شبكي)
   - الزر الآن في أسفل البطاقة
   - لون الزر: ذهبي (Dark) / أزرق (Light)

2. **إصلاح هيكل HTML**:
   - تم إصلاح `</div>` ناقص في قسم العروض
   - تنظيف الـ inline styles الزائدة

3. **تحديث ألوان الأزرار**:
   - زر "عرض التفاصيل": ذهبي مع تدرج
   - زر "إتمام الشراء": ذهبي بدل الأزرق الغير متناسق
   - زر "أضف للسلة" في product.html: متناسق مع الهوية

4. **إخفاء القوائم عند فتح تفاصيل المنتج**:
   - إضافة `!productDetailView` لجميع شروط الأقسام
   - CSS overlay لصفحة تفاصيل المنتج
   - إغلاق جميع الأ عند فتح منتج

5. **تحسين لون اسم المشروع في البطاقة**:
   - المود الليلي: أبيض (var(--text-white))
   - المود النهاري: كما كان سابقاً (var(--text-on-accent))

6. **حذف تبويب الاستشارات من صفحة الإشعارات**:
   - حذف الزر
   - حذف بيانات الإشعارات
   - حذف log المتعلق بالنوع

7. **إصلاح شاشة البداية (Splash Screen)**:
   - إعادة صورة PNG الأصلية (تم التخلي عن SVG)
   - زيادة مدة العرض إلى 3 ثوانٍ
   - إضافة `height: 100dvh` لملء الشاشة على الموبايل

8. **إصلاح السكرول والقوائم السفلية**:
   - زيادة `padding-bottom` لـ `.page-content` إلى 200px
   - إضافة `min-height: 0` لـ `.page-content` و `.main-content`
   - حل مشكلة إخفاء زر "أضف للسلة" خلف شريط التنقل السفلي

9. **تثبيت التطبيق (PWA Install)**:
   - إضافة بانر عائم يظهر أسفل الهدر عند توفر التثبيت
   - رسالة توضيحية: "تثبيت بنيان على جهازك"
   - زر "تثبيت" ذهبي + زر إغلاق (×)
   - حفظ حالة الإغلاق في localStorage
   - إخفاء تلقائي بعد التثبيت أو في وضع standalone

10. **إصلاح مسارات PWA**:
    - تغيير مسارات `manifest.json` من `/bnn/` إلى `/bunean/`
    - تغيير مسارات `sw.js` cache من `/bnn/` إلى `/bunean/`
    - إضافة ملف `.nojekyll`

11. **ترتيب أزرار الحساب**:
    - نقل زر "تسجيل الخروج" تحت "الشروط والخصوصية" وقبل "مواقعنا"

12. **صفحة الشروط والخصوصية**:
    - حذف المحتوى النصي بالكامل (سياسة الخصوصية + الشروط والأحكام)
    - استبداله بـ "قريباً..." لحين إضافة النصوص لاحقاً

13. **إضافة النايف بار لصفحة الشروط والخصوصية**:
    - إضافة `bottom-nav.js` للصفحة
    - إظهار شريط التنقل السفلي مع تحديد "حسابي"

14. **تحديث هوية زر التثبيت**:
    - استخدام `var(--accent-gold)` بدل الألوان الثابتة
    - يتغير للون الأزرق في المود النهاري

### ملاحظات تدفق العمل
- **تسجيل شركة** ← قيد المراجعة ← موافقة/رفض من الداشبورد
- **نشر مشروع** ← قيد المراجعة ← موافقة + نشر / رفض + ملاحظات
- **إعلان** ← قيد المراجعة ← موافقة / رفض
- **رسالة/استشارة** ← تظهر في الداش ← مراجعة وتعديل وموافقة
- **عرض سعر** ← قابل للمراجعة والتعديل والموافقة
- **السلايدر** ← إضافة/تعديل/حذف/ترتيب كل سلايدر على حدة
- **روابط التواصل** ← إدخال روابط واتساب، فيسبوك، انستغرام، تيك توك من الداشبورد وتظهر تلقائياً في صفحة الحساب
- **جميع الإجراءات** تتم داخل الداشبورد قبل النشر

### التقنية
- **حالياً**: `localStorage` مؤقتاً للواجهة الأمامية
- **مستقبلاً**: ربط بقاعدة بيانات حقيقية + API
- **الداشبورد** صفحة منفصلة بتسجيل دخول مشرف مستقل

## ملاحظات تقنية

- **لا يوجد backend حقيقي**: جميع البيانات في localStorage
- **التطبيق PWA**: يعمل بدون إنترنت + زر تثبيت عائم
- **متوافق مع الأجهزة المحمولة**: تصميم responsive
- **RTL**: واجهة عربية من اليمين لليسار
- **Vue 3 via CDN**: بدون بناء (build)
- **SCSS**: تم ترجمته إلى CSS يدوياً
